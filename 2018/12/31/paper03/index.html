2018 年终总结

发表于 2018-12-31 | 分类于 随笔

2018 最后一天，回顾去年写过的一篇文章，关于**我的 2017 年个人总结**执行情况：

1.技术

对 Java 语言更进一步的深入学习，另外学点 Python、HTML5 ，掌握一些最基础的开发工作。

完成情况：这个目标完成得很糟糕。最主要的原因是没有合理分配好时间。工作之外，去掉每天学习英语、阅读、其他等学习所花费的时间，剩下来的学习时间非常有限。

2.投资自己

花时间花钱大量的阅读，写作、实践，让自己从技术、写作、认知、思维等各方面不断提升。

完成情况：比较满意。无论是视野上的开拓，还是认知与思维上的改变，这一年的收获是巨大的，但在写作输出方面还需要很大的提升，虽然尽量保证了一点质量，但输出频率比较低。

3.英语

希望通过明年的坚持，更早的脱离国内的一些翻译过的技术文档资料，自己能够看懂一些英文版技术文档等，第一时间获取一些最前沿的技术知识。

完成情况：比较满意。今年的目标是学完新概念英语一册，虽然学完后，现在去阅读一些英文技术文档还是有些困难，但总算坚持做成了一件事，把这一套教材完完整整地学完了，成就感十足。

4.结婚

为人生当中的第一件大事提前做好一些准备。

完成情况：整体上来说准备的还不够充分，自认为没有做到最好。主要原因在于自己，婚前准备是两个人的事，大大小小的一些事都是需要花很多时间精力投入进来的，这些其中很多都是女朋友在操心，自己投入的时间非常有限。

5.努力赚钱

钱赚多了，很多事情就变了。

完成情况：和去年一样，公司今年也没有年终奖。好在薪资涨了一点，但口袋还是一样空空的。

先说说工作方面，今年初，刚开始的时候还是负责原来的几个项目，之后，团队成员人数上有了较大的变化，由原来的六个组成员到现在只剩下自己一个了，所有安卓相关的一些项目，也都由自己一个人负责。所以需要承担的责任也更多了一些，同时个人能力也有了较为显著的提升。刚开始接手项目时会遇到很多问题，但好在短时间内通过快速地学习解决了大部分，确保了项目顺利完成并如期上线。另外同时也对多个项目上反馈的一些问题及时处理，以及不定期的项目框架、基础库升级，包括后面项目持续集成环境的部署及维护等。

通过工作业余空闲时间，大量的阅读了关于 IT 互联网行业相关的各方面知识，开拓了视野的同时对于认知与思维方面也有了较大的提升。也许现在投入的大量时间阅读，并没有给自己带来什么实质性的变化，但或许就在未来的某一天，需要面对一些方向性选择的时候，所带来的影响一定是非常大的。

另外工作业余空闲时间，通过印象学院，报名学习了关于知识管理方面的课程，通过平时生与工作中不断地进行了实践尝试，收获颇大。 说来这个课程原本不在今年的计划范围内，由于每天的阅读量较大，所花费的时间和精力非常有限，对于如何筛选出优质的信息来源来提高阅读质量，以及如何更好的去消化、整理分类等来提高阅读效率来说是一件非常重要的事了。正好一直长期都在使用的印象笔记官方里面推出了与这些相关的课程，并且长期关注的某知名大 V 也推荐过，于是很果断的进行了学习。

课程里面还包括讲解了一些好的学习方法、个人任务管理方法等，比如费曼笔记、GTD 等。在学习的过程中，也见识了更多以前没用到过的新的效率工具，并运用到了生活与工作方面中。

……

再来说说生活方面，最难过的是今年的眼睛近视度数上升较为明显（大概五十到一百左右），算是有得有失吧，最主要的原因是长时间的阅读造成的，忽视了对眼睛的保护。

关于身体锻炼方面，今年下半年办的健身卡，主要是陪女朋友一块的，对于长时间对着电脑久坐不动，适当的健身运动下好处还是挺多的。每周大概去锻炼两次左右，没有请健身教练，基本上是过去打打酱油按照自己喜欢的方式锻炼了。持续了三个月不到，效果并不明显。到了国庆过来后的两个月一直在加班，后面基本上就再也没去过健身房了……

关于旅游方面, 今年清明节假期在杭州周边找了个相对人比较少的风景区（桐庐县）和女朋友一块度过了三天，去过荻浦花海，深澳古村、大奇山森林公园。短暂的几天停留，过的确是无比的充实，瞬间就将平时工作中的疲惫抛在脑后，让人感到特别的放松，心情更是愉快。

关于电影，相比去年，今年去电影院看电影的次数少了很多，总共去过五次。基本上都在上半年了，下半年相对来说各种事情越来越多了。

关于饮食方面, 和往常一样，平时周末有空都会尝试去做过一些新的菜式，辣子鸡等、红烧排骨、剁椒鱼头、豌豆玉米粒等。

……

2019 年，我想做的一些事：

  1. 登记结婚。继续为结婚做一些准备，希望婚礼办的比较顺利，更加完美，不留遗憾。

  2. 提升技术。安家立命之本不能丢掉。详细的目标计划这里就不列出来了，明年等实现了再说。

  3. 阅读与写作。减少阅读输入的同时，多一点思考，提高博客与公众号写作输出。

  4. 拥抱变化。能够应对工作或生活上带来的一些糟糕的事情。

最后，向所有阅读这篇文章的朋友说一声，新年快乐！
