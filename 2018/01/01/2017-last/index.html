我的 2017 年个人总结

发表于 2018-01-01 | 分类于 随笔

一转眼，2017 年已经过去了，在敲代码和改 Bug 中，又搬了一年砖，写点这一年的总结。

工作相关

春节过后，加入了一家创业公司歪脸网，在这一年中主要工作是参与「 歪脸助手 、再歪一点 、歪脸侠 」Android 端的开发和维护。

再歪一点刚进公司负责的一个旧的项目，项目代码非常臃肿，在进行维护与二次开发过程中进行了一些优化。对于性能优化方面有了一定的提升。

歪脸助手第二个参与的项目，负责其中一个模块的开发，整个项目开发了 4 个月周期，所有功能基本都已完成，但由于产品定位不明确等问题，放弃了上线，开发过程中熟悉了自定义控件等一些相关技术。

歪脸侠第 3 期版本开始参与的一个新的项目，项目里引用的大量第三方开源库较多，基本上都能熟练使用。欠缺的是对于源码方面的了解还不够深入，只知其然而不知其所以然。后面需要更多时间去阅读源码，希望在其中能吸收到一些好的设计与好的代码写法。另外在版本不断的迭代过程中，解决问题能力有了较大的提升。

学习相关

1、git 版本管理，熟悉使用命令行做代码管理；

2、报名了一些线上写作课程，对于写作方面有了一些认识；

3、英语语法方面有了进一步的提升，能够看懂一些简单结构的语句；

4、掌握了基础的 MarkDown 语法进行文档写作，对文案排版方面有了一定的提升；

5、搭建了个人博客，够买了域名，并开通了个人微信公众号，了解了一些运营方面的知识；

6、大量的阅读，思维与认知方面有了很大的变化。主要阅读来源：公众号( stormzhang、MacTalk、小道消息、caoz 的梦呓、Spenser )、知识星球、得到、知乎、36 氪、简书、腾讯新闻等；

7、熟练一些常用的效率工具，时间管理「 aTimeLogger2 」，专注工作「 Forest 」 ，思维导图「 XMind 」，写日记「 DayOne 」，知识管理「 印象笔记 」，事务清单「 奇妙清单 」，记账理财「 口袋记账 」。

生活相关

1、年底在武汉买了一套房，完成了人生中的一件大事。同时也意味着还贷生活提前到来了，也许有点压力的人生才算完整吧~

2、年初 3 月份和女朋友云南的一次旅游，很有意义的一件事。听过古城最朴质的倒鼓音乐声，见过泸沽湖清澈见底的湖水，了解过一些少数名族文化，去过风花雪月的洱海等。世界那么大，如果有机会，尽可能的多出去走走，也许能发现属于自己不一样的天空。

3、今年又学做了几道新菜，爆炒小龙虾、甲鱼汤、清蒸河虾、酸辣毛豆角，清蒸鲈鱼、玉米胡萝卜排骨汤等。

感悟

1、知识付费

在经济能力允许的情况下，只要能提高效率，节约时间，能解决问题，就花钱。

2、思考

在忙也要留给自己一些思考的时间，别用行动上的勤奋去代替思维上的懒惰。

3、时间

珍惜别人的时间，也是对别人的一种尊重。不同的人时间成本不一样，也许你还没到别人那个阶层，现在还无法体会到，浪费别人的时间也许就是在谋财害命。

4、社交

通常我们建立且维持人脉最常见的方式是保持联系不断的跟人交流， 但随着数量的增加，每增加多一个熟人情感联系，就要投入更多的时间精力成本，而随着数量的增加，10 个 20 个或许勉强还能维持的住，再往上如果是 200 个呢？很就有天花板瓶颈了。于是今年决定放弃了所有的社交，把几乎所有的时间和精力都投入到了写作这件事上，希望能锻炼下自己文字的表达能力，并积累一点影响力，同时也希望通过写作传播一点有价值的东西，通过这些知识点或互动反馈等建立情感联系，也让更多的人主动去认识我，且不给别人的时间成本造成负担。

5、专心做好一件事

有时候我们想着同时做好很多件事，做事很浮躁，到最后都没有一件事做成的，越是焦虑，越是要沉下心来，专心做好一件事，当一件事做完后再去做另外一件事，就会缓解了一部分焦虑，焦虑也就越来越少了。

6、如何高效学习

简单举个例子拿学习英语来说，其实死记硬背 8 个小时一般都不如间隔的去学习 4 个小时。比如每天学习半个小时英语，学习 8 天时间。原因是大脑在你学习间隔期能够自动复习学到的信息。所以提前规划好每天，学习一点点，不仅花费的时间少，而且学习时更加高效。

7、如何更好的去坚持学习？

最好的学习方法是在实践中学习 。一开始先要有个目标，然后针对这个目标去进行有针对性的学习。拿我今年自己做的一些事情来说，最初的目标是想在博客上写点有价值的东西分享出来，首先自己尝试搭建了个人博客，准备开始写点东西时根本就无法动手，感觉没什么可写，接着就在网上搜了下怎么写博客啊，如何提升写作技巧啊等方面的资料，最后决定了报名一个线上写作课，

在学习了一些写作相关基础知识后，于是带着目标的进行大量的阅读思考等，看看别人怎么写的，有哪些写的好的文章或句子值得自己学习或思考的，收集后通过知识管理工具整理下来等，最后当你回头再去写点东西时，你会发现自己能写出来有价值的东西比一开始的时候多了很多。在这些一个个小目标学习过程当中，当你每学一点新的东西，有了一个正面的反馈，你会获得一点点成就感，这样才会更容易的坚持下去。

明年计划

1、技术

对 Java 语言更进一步的深入学习，另外学点 Python、HTML5 ，掌握一些最基础的开发工作。

2、投资自己

花时间花钱大量的阅读，写作、实践，让自己从技术、写作、认知、思维等各方面不断提升。

3、英语

希望通过明年的坚持，更早的脱离国内的一些翻译过的技术文档资料，自己能够看懂一些英文版技术文档等，第一时间获取一些最前沿的技术知识。

4、结婚

为人生当中的第一件大事提前做好一些准备。

5、努力赚钱

钱赚多了，很多事情就变了。

最后，祝大家新年快乐！
