我的 2019 年个人总结

发表于 2019-12-31 | 分类于 随笔

和**去年**一样，记录下今年的个人总结。

先来说说工作与生活方面，再来回顾年初目标完成情况，最后列举下新年的计划目标。

工作方面

在维护线上核心的一个项目基础上，中间完成并加入了一些新的业务模块，确保产品的正常迭代。与此同时，开始为新的项目做一些准备。比如：

  • 根据项目当前的整个业务需求以及后期版本迭代可能出现的业务需求，从零开始着手搭建整个项目的技术框架。

  • 对当下热门新技术的可行性进行测试，确保没有其他问题后，并引入到项目中。

  • 在团队项目开发过程中，可能会写很多重复性代码，针对这些，编写了各种工程模板，比如代码块模板（Live Templates）、单个文件模板（File and Code Templates）、和整个工程模板（Plugin Templates），为整个团队开发节省时间。

  • 提前制定好项目开发规范文档，给后面新进来的团队成员开发做好准备，确保整个项目规范的统一性，便于日后项目的可维护性。

  • 由于整个项目过于庞大、功能模块较多，基于公司对项目周期的严格把控，从年初开始，负责招聘新的团队成员。经过多份简历的不断筛选与面试，最终引进了二到三个新的团队成员。通过将任务拆解并分配到合适组成员后，项目开发从五月份开始，期间不断加班，一直持续到现在。开发期间伴随着一系列大大小小的需求增改等，到目前为止，项目仍未上线。

和去年对比，今年在工作职责上有了一些变化，对自己有了一定的成长与收获。从年初开始，不断的在面试招人，在面试的过程中也发现了自身一些不足的地方，同时也结识到了几个有趣的朋友。另外在主导项目过程中，对组成员之间的协作能力有了一定的提升。

另外还有些想说又不太想说的话，留着以后再说吧。

生活方面

  • 关于结婚。今年的国庆节，我们结婚啦。把人生的另一件大事给办了，刚好这也是以前自己期望的一个结婚年龄。在没办之前，平时工作时心里时而会惦记着这个事，难免还会有些分心，现在不用再惦记了，可以专心的去做其他事了。

  • 关于旅游。国庆节办完婚礼后，去西藏玩了半个月，写了西藏，一辈子一定要去一次的地方，返回时在西安玩了三天，参观了另一处世界文化遗产「秦始皇兵马俑」。这一次的旅游过程虽然有些疲倦，但也收获了很多。从一个熟悉的地方去往另一个完全陌生的地方，不仅开阔了眼界，在锻炼自身适应能力的同时，通过见识到不同的人与事物，也进一步培养了自身的同理心。

  • 关于电影。这一年总共看了十三部电影，按时间顺序依次为：大黄蜂、流浪星球、比悲伤更悲伤的故事、复仇者联盟 4、绿皮车、蜘蛛侠 - 英雄远征、狮子王、哪吒之魔童降世、海贼王 - 狂热行动、攀登者、烈火英雄、中国机长、千与千寻。其中标识下划线的为电影院观看，其他为电脑上观看。「攀登者、中国机长」这两部刚好在国庆期间上映，从西藏回来后看完的，在里面看到了刚去西藏见过的珠峰大本营和珠穆朗玛峰，还有布达拉宫等场景，假设性的将自己代入到电影中去体验，看完后颇有感触。

  • 关于饮食。由于今年长时间的加班，在外面吃饭的次数也多了些。在周末或下班后也尝试做了一些新菜，如：清蒸螃蟹、毛血旺、烤鱼、蒸粉肉、麻辣香锅、椒盐皮皮虾等。想起以前下班的时候，虽然有些忙碌，但还是能挤出时间回到家边做下饭炒炒菜，可以静心，一边还可以想想工作或生活上的一些事，感觉挺好。

  • 关于健康。虽然这一年有一大半时间都在加班，好在身体还能扛得住。年底前做了一次体检，身体状况良好。但眼睛视力明显感觉有所下降。虽然年初时配了副眼镜，但还没有培养成经常戴的习惯。明年需要再做一次视力检查，做好预防；另外在今年的最后一个月，拔完了剩下多余的三颗智齿，可能是有那么一点强迫症的原因，长着多余的东西，拔完后心里踏实了很多，不用再为偶尔牙痛发炎的事而烦恼惦记着。

回顾年初目标完成情况

2019 年初，计划完成目标：

  • 登记结婚。继续为结婚做一些准备，希望婚礼办的比较顺利，更加完美，不留遗憾。

  • 提升技术。安家立命之本不能丢掉。详细的目标计划这里就不列出来了，明年等实现了再说。

  • 阅读与写作。减少阅读输入的同时，多一点思考，提高博客与公众号写作输出。

  • 拥抱变化。能够应对工作或生活上带来的一些糟糕的事情。

第一个目标已经完成。毕竟在这上面的准备也花费了不少时间，婚礼办的还算顺利，也没留下什么遗憾。当然这个是媳妇说了算。

第二个目标算是完成，但还远远不够。比如对 Kotlin 语言有了更进一步的掌握，并很好的运用到了工作场景中。另外学了些脚本语言 Python、JavaScript 简单语法。其中 Python 语言这也是在 18 年初制订的一个目标，但因为一些原因没有合理分配好时间，后来给放着了。今年通过学习对 Python 语法掌握了一些，但因为缺少了实践使用场景，没有很好的运用到工作场景中，渐渐的学习一段时间后很快就给遗忘了。总体上来说，在技术上没有什么较大的突破。

第三个目标只完成了一半。阅读输入量不但没有减少，反倒增加了很多，但还是缺少了一些自己的总结和思考。惭愧的是一年到头来扳着手指都能数得出来，总共写了三篇公众号文章，这个输出频率远远还不够。原因可能是缺少了一点让自己坚持下去的动力吧，阅读量少，关注量也少，更不好意思厚着脸皮拉好友来关注，没有留言互动，就这样越是没有反馈，就越是拖延输出，形成了一个恶性的循环…… 后面需要提高输出频率，想到的首要解决办法是，先给自己下一个明确的输出目标规定，等执行一段时间后，开始不要脸的先去拉点好友关注，慢慢有了一些互动反馈后，就会更有动力的坚持下去，形成一个正向循环……

最后一个目标，生活上其实没有什么变化，但工作上面确实有了些变化，这也是年初计划意料之中的事。提前做好准备，保持一个好的心态，就能够轻松应对，更好的继续向前。

2020 年，计划做的一些事

  • 保持 Android 开发方面能力的同时，扩展 Web 前端技术。

  • 坚持一到两周发布一篇博客或微信公众号文章。

  • 学习英语是一个长期积累的过程，不进则退，不能丢下。养成习惯，每天花十分钟时间学习，重要的是坚持。

  • 锻炼身体，养成每天跳绳五分钟的习惯。无意间看到刘润老师写的一篇文章 如何把一天用到极致？中有提到：“ 从训练心肺功能角度来看，跳绳 5 分钟，相当于慢跑半小时。姿势正确的话，跳绳对膝盖的伤害只有跑步的七分之一。”，同样是训练心肺功能，但跳绳却比跑步的效率要高很多。原本打算从跑步开始坚持锻炼的，却发现对跑步的认识其实太少了。


最后，在即将到来的 2020 年送给自己一句话：“ 不仅仅要正确的做事，更重要的是做正确的事。”，与你共勉。

感谢一路过来所有遇到过的人，无论是好的坏的，都让自己有所成长。

感谢一路过来给予我支持帮助过的人，有你们的支持和帮助才会有现在更好的自己。

谢谢你们！
