我的 2020 年个人总结

发表于 2021-01-18 | 分类于 随笔

2020 年还没来得及说声再见，2021 年已经悄悄过去了半个月。和往常一样，记录下这一年我的个人总结。

这一年，一个个传奇人物因各种意外而离开，比如美国篮球巨星科比·布莱恩特、阿根廷传奇足球明星迭戈·马拉多纳、美国鞋类电商网站 Zappos 创始人谢家华等；这一年，很多中小企业生意惨淡，各种裁员、各种倒闭；这一年，新冠疫情、澳洲大火、东非蝗灾、英国脱欧、美股熔断……

另外一边，电商直播、短视频、知识付费行业发展如火如荼，北斗导航系统全球组网完成、RCEP 自由贸易协议签署成功……

这一年，世界在悄悄的改变。

下面来说说，在这魔幻的一年，关于一个普通打工者的生活记录。主要分为：工作方面、生活方面、个人 Milestone，回顾 2020 年初目标完成情况，2021 年我的计划目标，与你分享我的几点感悟。

工作方面

目前主要参与的一个跨平台 ReactNative 项目，从涉及到的技术栈方面来看，对我来说这是一个全新的项目，可以接触到一些 Web 前端技术，比如项目中使用的 JavaScript 语言和 ReactNative 应用开发框架等，这些都在去年规划的目标方向上。

虽然应对工作日常的一些开发任务，没有太大难度。但仅仅只是停留在会使用的地步远远不够，还有很多不足与挑战的地方，比如 ReactNative 性能优化方面、底层框架 React 的学习理解，另外涉及的一些 Web 前端技术等。除了代码技术外，业务也需要花时间去学习。

生活方面

关于支出， 全年总共支出金额￥83635 元（不包含房贷和媳妇自己的一些支出），和去年总支出相比，增加了10.24%。其中效率/学习提升相关支出占总支出比例 41.26%，生活费相关支出占总支出比例 58.74%。

关于健康， 今年眼睛视力又下降了一些，和去年相比，现在工作时几乎每天都在戴着眼镜了；体重方面，现在 115 斤，属于偏瘦型的，从过去五至十年来看，从来没有超过 120 斤，只要保持健康饮食习惯和作息规律，就不用担心后面会长胖的事了，未来十年到二十年可能都是这样。

关于饮食， 今年在外面吃饭的次数多了一点，中间也尝试了一个多月的代餐饮料「若饭」，经过亲身体验，没有很多人担心的一些副作用，比起经常点外卖和在一些店里点餐，其实更加健康、营养也更全面，主要还方便省时。另外和往年一样，今年解锁的新菜有：红烧鱼块、酥炸小鱼、炒田螺、地三鲜。经过这几年的一点积累，截止到 2020 年止，总共解锁了刚好 20 道新菜。

关于影剧， 今年看过的影剧很少了，主要陪媳妇一起看的，比如盗墓笔记、鬼吹灯。另外看过两集脱口秀节目，还有几部电影，比如哥斯拉2、金刚川、八佰、叶问4。

关于旅游， 受疫情影响，今年没有旅游计划，所有就只有杭州本地游了。11 月份去过一个 5A级景区灵隐寺，在济公的出家地许了个愿望，不知道会不会显灵哈：）；还有周边的大明山玩了一天，因为时间关系，还有个滑雪场想去玩的没有体验到；最后是西湖景区，虽然都去过很多次了，但今年在夕阳偶然路过的一次，却拍到了我认为的最经典最好看的照片。

个人Milestone

开源项目， 出于对新技术学习与整合的目的，以及对开眼 App 风格的喜爱，在去年六月份发布了一款高仿 Eyepetizer | 开眼短视频的 MVVM 开源项目 ，截止到目前为止，将近半年时间左右，总共收获了 886 个 Star，这里也要感谢一些公众号作者、博主、朋友的热心推荐，让这个项目能在刚开始起步的阶段，得到了很快的传播。在项目刚发布的前一至两个月期间，便陆续霸占了 GitHub 热门榜单(Kotlin)：日榜 Top 1、周榜 Top 1、月榜 Top 3。另外一个值得高兴的事，在开源项目发布了三周后，在 Issues 上看到了一条标题为（开眼招聘）关于 Android 开发岗位的招聘信息，间接的也算得到了开眼官方对这个项目的认可。

链接， 通过互联网，与不同的人建立了链接。

  1. 猫哥，一位 47 岁家住陕西，正在国企电力工作已经长达十年的大叔，业余时间还在自己折腾给公司开发 App ，因为 Eyepetizer 开源项目，我们结识。

  2. 微信开发团队，通过在 Mac 端微信内测版上 ，提了一些日常使用中遇到的 bug 意见反馈后，通过微信客服，与某个微信开发团队小组产生了链接；

  3. 郭霖，通过某微信社群，主动添加了郭霖老师个人微信，并有了一次简短的交流，之后在线下参加了上海 GDG 技术会议，并见到了郭霖老师本人，并在面对面有了更进一步的交流，返杭前，有幸和郭老师合了个影；

  4. Fenng（冯大辉），在知识星球社群，因为视频号（互联网消息）取名受到冯大辉老师启发，在表达了一番对冯大辉老师的感谢后，我的视频号很幸运的受到了冯大辉老师关注。之后又有几次通过冯大辉老师公众号（小道消息）在后台留言互动，产生了链接。

  5. 二爷（邱岳），通过在二爷的某个产品公开课直播间，一次偶然的互动，让二爷注意到了，之后便在微信上产生了链接。

  6. 生财有术球友，通过生财有术知识星球社群，链接了一帮会赚钱的人，涵盖各个行业不同领域。

  7. 其他，还有各种各样不同的人，比如，有初入职场的小白、也有行业资深专家、还有影响力颇高身价千万过亿的大佬等。

微信， 新增联系人数量 1500+，目前总共联系人 2080，其中技术占比例 50%，其他占比例 50%，主要包含产品、运营、设计、数据、销售、游戏、市场、创业等；订阅的公众号新增 300+，目前总共订阅公众号数量 508 ；群聊新增数量 100+，目前总共群聊数量 135。

知识星球， 总共加入了 31 个知识星球社群，其中付费知识星球占比 54.9%，免费占比 45.1%。星球涉及领域主要包含：产品、技术、赚钱、职场、房地产、运营、视频号、商业、营销、数据分析等领域。

回顾 2020 年初目标完成情况

  1. 保持 Android 开发方面能力的同时，扩展 Web 前端技术。

完成情况评分：70 分。通过 Eyepetizer 开源项目，对学习 Android 一些新技术进行了整合，另外通过公司的跨端 ReactNative 应用框架开发项目，扩展了一点 Web 前端技术，目前停留在可以完成简单的界面交互阶段。

  1. 坚持一到两周发布一篇博客或微信公众号文章。

完成情况评分：50 分。全年微信公众号总共更新了 7 篇文章，其中有 4 篇是转载的。每次准备开始写点东西的时候，写着写着最后觉得太过于简单，无法给读者提供价值，以至于写着写着就没然后了。于是后面采用了一种算是迂回的办法，尝试先在朋友圈进行一些简单的分享，如果连朋友圈都坚持不下去，又怎么能在公众平台写文章坚持下去。后面就在朋友圈开始了实践，一年下来总共发了约 231 条朋友圈，平均每周 2~3 条。主要输出内容有：互联网行业相关资讯与见解、分享优质文章、好物推荐、个人生活状态等。

  1. 学习英语，每天花十分钟时间学习。

完成情况评分：16 分。年底最后两个月，在百词斩 APP 连续打卡积累单词量；

  1. 锻炼身体，养成每天跳绳五分钟的习惯。

完成情况评分：25 分。疫情期间，在家坚持了一个月左右跳绳运动，回杭州后接着持续了两个月的跳绳运动。

这一年，对我来说，有好也有坏。

不好的一方面，在这一年中，整个人的心态相对来说比较浮躁，对于技术方面的专注，缺少了以往的一些耐心，一方面是受疫情的原因影响，另外一个原因可能是今年所关注的东西比以往更多了一些。

好的一方面，主要有两点：1.对信息的处理效率又有了进一步的提升。首先是信息源的变化，新增了多个优质的付费星球社群、微信群聊，朋友圈的升级。其次是对信息的快速筛选去除，删掉与屏蔽一些无关紧要的人、信息，从原来无意识的被动获取信息到现在有意识的主动获取自己需要的信息，最后经过加工整理后输出到朋友圈。2.链接了各行业不同领域优秀的一群人，通过他们不断的刷新了自己的认知，极大开拓了眼界，对于未来可能会产生的深度链接，也增加了一种可能。

2021年我的计划目标

今年的计划目标只有一个，读书。一年下来，看看能读多少。有时目标订得过多又总是没能达到预期结果，反倒无形的给自己带来了一种压力和负担？不如先试试订一个目标看看，明年再看结果。如果时间还够的话，把豆瓣电影 Top 50 剩下还没看完的，都给看了。

与你分享我的几点感悟

  1. 价值观。引用『宫崎骏—千与千寻』里的一段话：“人生就是一列开往坟墓的列车，路途上会有很多站，很难有人可以自始至终陪着走完。当陪你的人要下车时，即使不舍也该心存感激，然后挥手道别。” ，让那些与你价值观不同、性格不合的人，慢慢在时间的流逝中一点点消失吧。

  2. 赚钱。如果穷也是一种病，那就努力赚钱吧。如果你有能力，或许还可以让你关心的人再延续多活几年。钱虽然不是万能，但钱可以治疗大部分问题，你对钱的欲望有多强烈，你就有多大的动力去赚钱。如果你很迷茫，不知道活下去的意义，不知道自己要什么，那就努力地去赚钱吧，至少不会在你关心的人需要你的时候而无能为力。

  3. 照顾好家人。在能力允许范围内，尽可能的多给家人一些帮助，物质方面的也好，精神方面的也好，尽可能的给他们/她们提供最好的生活条件，多给他们/她们打打电话，多陪他们/她们聊聊天。把时间浪费在家人身上，浪费在生命中最重要的人身上，回头来看，这也是我最近这五年来做的最正确的一件事。

最后在 2021 年许个愿望，希望疫情尽早结束，每个人都可以尽快的恢复到原来正常的生活节奏。是太阳先升起，还是意外先来临，对于现在来说，活下来就已经是一件幸运的事了，保持一个好的心态去面对接下来的一年，永远相信美好的事情即将发生。

万物之中，希望至美。至美之物，永不凋零。
